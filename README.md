# product-profile
This repository provides an overview of methodologies used for Product Healthiness Profile assessments by ATNi over time. It serves as a resource for understanding how nutrient profiling models have been applied by ATNi in its Global Indexes, country level assessments and thematic assessments. 
# What is Nutrient profiling
Nutrient profiling is the science of classifying foods based on their nutritional composition to prevent disease and promote health. It helps identify which foods contribute to a healthy diet and supports policies that influence consumer choices and food formulation.
